# Guía de Contribución

Gracias por considerar contribuir a Hotel Paraiso. Aquí hay algunas pautas para ayudarte a empezar.

## Cómo reportar errores

Antes de crear reportes de errores, revisa la lista de issues para asegurarte de que el problema no haya sido reportado anteriormente.

Cuando crees un reporte de error, incluye:
- Una descripción clara del problema
- Pasos específicos para reproducir
- Ejemplos específicos para demostrar los pasos
- Descripción de lo que observas y qué esperabas ver
- Screenshots o videos si es posible
- Tu entorno (navegador, SO, versión de PHP)

## Proceso de desarrollo

1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/AmazingFeature`)
3. Commit tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. Push a la rama (`git push origin feature/AmazingFeature`)
5. Abre un Pull Request

## Estándares de código

- Mantén consistencia con el código existente
- Comenta el código cuando sea necesario
- Asegúrate de que el código sea responsive
- Prueba en diferentes navegadores y dispositivos

## Estructura de carpetas

Mantén la estructura de carpetas organizada:
- `admin/`: Páginas públicas del hotel
- `css/`: Estilos globales
- `js/`: Scripts globales
- `style/`: Estilos adicionales

## Testing

Prueba tus cambios en:
- Chrome/Chromium
- Firefox
- Safari
- Edge
- Navegadores móviles
